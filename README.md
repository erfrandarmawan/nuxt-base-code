## Requirements
- Node version 22

## Setup Project
- Install node packages
run command `npm install --verbose`
<br>

- Copy file `.env.example` into `.env`
<br>

## Run Project
- run command `npm run dev`
<br>

## Build Project
- run command `npm run build --verbose` or `npx nuxi build`
